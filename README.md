# Aurora RAG Chatbot


## Quick Start

```bash
pip install -r requirements.txt
./start.sh
```

Open: http://localhost:8000

## Dashboard

Access: `/dashboard`

Credentials are in `.env`

## Environment Variables

Create `.env` file:
```
GROQ_API_KEY=
GOOGLE_SHEETS_ID=
DASHBOARD_USERNAME=
DASHBOARD_PASSWORD=
```
